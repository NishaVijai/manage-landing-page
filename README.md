## Description

**Manage Landing Page** is a **responsive marketing landing page project** built using **semantic HTML and SCSS**, based on a **Frontend Mentor challenge**.

The project demonstrates a **modern landing page layout**, including hero section, features, testimonials, and call-to-action elements, emphasizing **responsive design**, **CSS Grid & Flexbox**, and **pixel-perfect implementation**.

2. **Open the project**

No build tools required.

```bash

```
